# vpn_project

Этот Telegram-бот предназначен для оформления и управления подпиской на VPN. Пользователь может выбрать тариф, оплатить подписку через банковский перевод, получить конфигурацию двумя способами, а также управлять устройствами: добавлять новые и удалять старые. Бот автоматически уведомляет о скором окончании подписки и позволяет быстро продлить её.

##  Функциональность

- Создание базы данных и таблиц
- Клавиатуры 
- Обработка скриншотов
- Подтверждение подписки админом
- Выбор устройства и выдача инструкции для этого устройства
- Создание конфигурации на сервере
- Отправка конфигурации клиенту (через файл или QR-code)
- Фоновая проверка на окончание подписки (2 раза в сутки)
- Управление подписками (добавление новых устройств и удаление старых)
- Продление подписок
- Вывод профиля клиента со всеми подписками

##  Технологии

  - 'sqlite3' 
  - 'threading'
  - 'wg_easy_api_wrapper'

  - 'logging'
  - 'asyncio'
  - 'aiohttp'
